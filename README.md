Dataset
This project uses the METABRIC breast cancer dataset.
Source Kaggle
https://www.kaggle.com/datasets/raghadalharbi/breast-cancer-gene-expression

The dataset is a processed version of the original METABRIC study and includes gene expression data, clinical features, and PAM50 plus Claudin-Low subtype labels.
Access requires a Kaggle account.
The Kaggle API key file kaggle.json is not included in this repository for security reasons.
